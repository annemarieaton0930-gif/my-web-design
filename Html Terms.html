Html Terms


Page Structure Tags

html
It is a root element for an HTML document. All elements are placed inside this element.
head
The head tag may contain scripts and styles that are useful page functionalities. Meta tags are also placed inside this tag.
body
It defines the body of the webpage, all elements that are used to display content on the browser placed inside the body tag.
!DOCTYPE
A document type declaration that is used by the browser to understand the version of the HTML used in the document.

Formatting Tags

h1 to h6
Heading tags used to define titles and subtitles.
p
Defines a paragraph of text.
br
Inserts a line break.
hr
Creates a horizontal rule or divider.
style
Refers to using the style attribute or CSS to control the appearance of elements.

List Tags

ul
Defines an unordered bulleted list.
ol
This is the type of list where each item is numbered usually starting at 1 (but can be changed using the start attribute).
li
Represents a list item inside ul or ol.
dl
It is an effective tool for making your own “definition of terms” list or glossary.
